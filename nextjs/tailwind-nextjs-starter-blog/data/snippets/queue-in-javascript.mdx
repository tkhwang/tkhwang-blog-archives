## âœ¨ using sliding window

```javascript

[0  1  2]  3  4  5  6  // sum
0  [1  2   3] 4  5  6  // sum + new (4) - old (0)
                       //     + nums[i] - nums[i-k]
```

- Instead of using a queue.
- precalculate value with fixed window size.
- Update value using sliding window
  - add the latest one `nums[i]`
  - remove the oldest one `nums[i-k]` from the previous result.

```javascript
const N = nums.length

let sum = 0
for (let i = 0; i < k; i += 1) {
  sum += nums[i]
}

let max = sum

for (let i = k; i < N; i += 1) {
  sum += nums[i] - nums[i - k]
  if (max < sum) max = sum
}

return max / k
```
