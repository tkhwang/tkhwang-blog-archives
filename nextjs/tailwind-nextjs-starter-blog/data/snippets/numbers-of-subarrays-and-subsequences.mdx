Count of possible **subarrays** using given length of Array

- The count of the subarray is the number of ends position possible for each index `i` elements of the array can be `(N â€“ i)`,

Therefore the count of the subarray for an array of size N can be:

```javascript
// N
 |-------------------|
 |-------|           |
 |---|   |           |
 |   |   |           |
 0   1   2  ...    N - 1
 1 + 2 + 3 + ... +   N


  N * (N+1)
= ---------
     2
```

```javascript
const countSubarray = (n) => (n * (n + 1)) / 2
```

Count of possible subsequences using given length of Array

- The count of the subsequence possible is each element of the array can be included in a subsequence or not.

Therefore, the choice for each element is 2.

```javascript
// N

2 * 2 * ... * 2 = 2^N
```

```javascript
const countSubsequence = (n) => Math.pow(2, n)
```
